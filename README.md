## 📌 Description

**New Post Type** est un plugin WordPress léger et efficace qui permet aux utilisateurs de créer et gérer des **types de publication personnalisés (CPT)** directement depuis l'administration WordPress. Ce plugin est conçu pour être simple d'utilisation, tout en offrant une flexibilité maximale pour répondre aux besoins spécifiques de votre site.

---

## 📦 Installation

### Méthode 1 : Installation via GitHub

1. Téléchargez le dépôt depuis GitHub :
  ```bash
   git clone https://github.com/Sta-ces/new-post-type.git
  ```
2. Compressez le dossier du plugin en `.zip`.
3. Dans votre administration WordPress, allez dans **Extensions > Ajouter**.
4. Cliquez sur **Télécharger une extension**, sélectionnez le fichier `.zip` et installez-le.
5. Activez le plugin.

### Méthode 2 : Installation manuelle

1. Téléchargez le fichier `.zip` depuis la [page des releases](https://github.com/Sta-ces/new-post-type/releases).
2. Suivez les étapes 3 à 5 de la méthode 1.
